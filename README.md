# 🎭 MoyuBrowser (摸鱼浏览器)

一个专为高压职场打工人设计的“终极防御型”悬浮视频浏览器。**没有任何多余 UI，没有任何穿帮特征，全功能靠盲操与全系统快捷键驱动。** 启动即看，移开即隐，给你的桌面留下最后的量子防御屏障。

> **🤖 联合出品声明：** > 本项目核心架构、内核级协议拦截、窗口无缝记忆及盲操逻辑，均由 **Gemini** 倾力提供底层技术支持与代码重构。

---

## ✨ 核心神仙功能

* **🧱 100% 纯净无框**：彻底移除了传统浏览器的地址栏、标题栏、控制条和边框。整个窗口只有纯粹的网页内容，外人看过来就像一个普通的桌面小挂件。
* **🦺 内核级弹窗绝杀**：强行重写了 Chromium 内核对 `bytedance://`、`snssdk1128://` 等流氓私有协议的认知，在应用内部直接静默超度，**100% 解决切视频时 Windows 闪烁 CMD 窗口、抢走输入焦点的行业痛点**。
* **🫥 边缘穿透与人间蒸发**：
    * **瞬间隐形**：鼠标只要点击应用以外的任何地方（如微信、Word），窗口在 0 毫秒内变为 **0% 完全透明，且鼠标直接穿透**，绝不耽误正常工作。
    * **量子复活**：按下 `Alt + S`，窗口在原地以你上次调好的透明度完美显现。
* **🏎️ 系统级盲操与精准控温**：
    * **盲操拖动**：鼠标按住窗口**左上角（顶部 1/10 高度，左侧 50% 宽度内）**即可直接拖走窗口，右侧其余地方皆为纯净网页，不妨碍任何点击。
    * **丝滑调光**：全系统全局快捷键驱动，无需激活窗口。按下 `Alt + ↑`（变清晰）/ `Alt + ↓`（变模糊），每次以 5% 步长精准微调透明度。
* **💾 智能闭眼记忆**：内置窗口状态追踪器。无论是手动调整、拖拽，还是按下 **`F` 键光速无痕火化（进程闪退）**，程序都会在后台无痕记录当前的横纵坐标和宽高，下次打开直接原地复活。
* **🍪 持久化持久登录**：内置独立会话隔离沙盒（Cookie 持久化），抖音只需扫码登录一次，后续打开永不掉线。

## 🛠️ 开发者本地构建指南 (克隆与自行编译)

如果你想在自己的电脑上运行此项目，或者将其编译打包成独立的单个 `.exe` 绿色便携版，请遵循以下傻瓜式步骤：

### 1. 准备工作
将本项目的所有核心源文件下载或解压到本地文件夹：
* `package.json`
* `main.js`
* `index.html`
* `preload.js`

### 2. 环境配置与构建
打开你的终端（CMD 或 PowerShell），切换到项目根目录下，依次复制并运行以下命令：

```bash
# 🔥 第一步：切换国内阿里镜像源（老手都懂，防止 Electron 依赖包下载卡死）
npm config set electron_builder_binaries_mirror [https://npmmirror.com/mirrors/electron-builder-binaries/](https://npmmirror.com/mirrors/electron-builder-binaries/)
npm config set registry [https://registry.npmmirror.com](https://registry.npmmirror.com)

# 📦 第二步：自动下载项目依赖
npm install

# ⚙️ 第三步：补齐本地打包开发工具
npm install electron-builder --save-dev

# 🚀 第四步：一键编译打包
npm run build
