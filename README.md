# 🎵 MoyuBrowser (抖音网页端极简轻量浏览器)

一个专为追求纯净体验的用户打造的抖音网页端极简悬浮窗浏览器。**移除了一切传统的浏览器边框与多余 UI**，支持全局快捷键控制与失焦无干扰模式，让你可以将视频画面以极简的状态固定在桌面任意位置。

> **🤖 联合出品：** > 本项目主进程架构、内核协议拦截、全局快捷键微调及窗口位置记忆逻辑，均由 **Gemini** 提供核心底层技术支持。

---

## 🛠️ 功能特性

* **📺 极简无框视窗**：彻底去除了地址栏、状态栏和窗口边框。启动后整个窗口 100% 充满网页内容，无任何多余元素的视觉干扰。
* **🛡️ 客户端请求拦截**：在 Electron 内核层直接接管并拦截网页端特有的私有协议（如唤起本地客户端的自定义链接），避免切视频时 Windows 系统弹出错误提示或闪烁黑窗口。
* **🌓 失焦无干扰模式**：
    * 当鼠标点击其他软件（窗口失去焦点）时，本浏览器会自动变为完全透明，并开启鼠标穿透，绝对不影响你正常的桌面办公和软件操作。
    * 按下快捷键 `Alt + S` 即可瞬间唤醒，在原地恢复原有透明度并重新聚焦。
* **🕹️ 顶部快捷拖拽**：窗口顶部 10% 高度的左半边为隐形拖拽区，鼠标按住此处即可随意移动窗口；其余 90% 以上的画面区域对鼠标完全开放，方便正常点赞或切视频。
* **🎛️ 全局快捷键微调透明度**：支持使用全局快捷键 `Alt + ↑` / `Alt + ↓` 调整窗口透明度，每次步长为 5%，方便根据当前的桌面背景调节到最舒适的视觉状态。
* **💾 窗口状态记忆**：退出程序或按下 `F` 键关闭时，系统会自动在后台记住当前窗口的坐标（X、Y轴）以及宽高（Width、Height），下次启动无需重新调整。

## 📦 本地开发与项目打包教程

### 1. 准备文件
确保你的本地文件夹中包含以下 4 个核心文件：
* `package.json` （项目配置与构建参数）
* `main.js` （主进程控制逻辑）
* `index.html` （窗口容器与拖拽热区划分）
* `preload.js` （网页预加载拦截脚本）

### 2. 编译构建步骤
打开命令行工具（CMD 或 PowerShell），进入该文件夹路径，依次运行以下命令：

```bash

# 1. 安装项目基础依赖
npm install

# 2. 补齐本地打包开发工具
npm install electron-builder --save-dev

# 3. 执行一键打包
npm run build
